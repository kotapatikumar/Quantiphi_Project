# Subscription Tracker & Renewal Dashboard

A web application that helps users manage and track recurring subscriptions such as Netflix, Spotify, ChatGPT Plus, and other SaaS services.

## Features

* Add and manage subscriptions
* Calculate monthly burn rate
* Track upcoming renewals
* Renewing Soon alerts
* Active / Paused subscription toggle
* Real-time savings simulation
* MongoDB data storage

## Tech Stack

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MongoDB

## Run Locally

```bash
npm install
node server.js
```

Open:

```text
http://localhost:5000
```
